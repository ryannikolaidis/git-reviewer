Installation
============

From Source
-----------

.. code-block:: bash

    git clone https://github.com/ryannikolaidis/git-reviewer.git
    cd git-reviewer
    make install

Development Installation
------------------------

.. code-block:: bash

    git clone https://github.com/ryannikolaidis/git-reviewer.git
    cd git-reviewer
    make install-dev

Requirements
------------

* Python 3.12+
